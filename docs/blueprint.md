# **App Name**: Akkane's Magical Boutique

## Core Features:

- Homepage Showcase: Display a visually stunning homepage with an animated hero banner, featured products, and an Instagram feed of @akkanesboutique.
- Shop Catalog: Shop page with filtering by Lolita, Gyaru styles, size, color and availability. Quick-view product previews with zoom on hover.
- Detailed Product Views: Product detail pages with multiple images, style tags, and a size chart modal.
- Brand Story: Display an 'About' page with scrollable illustrations telling the brand story.
- Contact Information: Present a contact page with a form, icons, links to social media, and store information.
- AI Style Suggestion: Use an AI tool to suggest style tags for new products.

## Style Guidelines:

- Primary color: Rose bonbon (#F070A1), a vibrant and nostalgic color suitable for this fashion brand.
- Background color: Light Pink (#F9D9E2), a desaturated tint of the primary for a soft, candy-like feeling.
- Accent color: Lavender blush (#FFDA77), offering a dreamy yet high contrast complement to the pink hues.
- Headline font: 'Playfair', a modern serif exuding high-end, fashionable elegance.
- Body font: 'PT Sans' sans-serif to complement Playfair, and offer readability for product descriptions. Note: currently only Google Fonts are supported.
- Illustrated icons include teacups, frilly frames, vintage mirrors, parasols, and bows in pastel or dark romantic shades.
- Employ scroll-reveal effects, shimmer hovers, and subtle sparkles for smooth and optimized performance.
